This step allows to customize the default kernel command line arguments.

The syntax accepted is the same as the kernel command line syntax, e.g. console=ttyS0,115200, but prefixing the argument with - removes the argument from the default command line (e.g. -console).

{{ if eq .SecureBoot "true" }}

With SecureBoot, the kernel command line is signed and cannot be modified, so this is the only opportunity to customize it.

{{ else }}

Please note, that the kernel command line customization is only accepted for the initial boot images (ISO, PXE, disk image), and ignored for installer images. For the upgrade/install command line customization, use the machine.install.extraKernelArgs machine configuration field.

{{ end }}

Skip this step if unsure.

{{ if .OverlayOptionsEnabled }}

This step allows to customize the overlay options for {{ .BoardMeta.OverlayImage }} overlay.

The syntax accepted is the YAML (JSON) key-value pairs. The list of options available is specific to the overlay image used, and can be found in the overlay image documentation.

Skip this step if unsure.

{{ end }}
